Εγκατάσταση του XOOPS

Εγκατάσταση για πρώτη φορά

Πρόλογος:

Το XOOPS είναι ένα σύστημα δημοσίευσης ιστοσελίδων ανοικτού κώδικα, αντικειμενοστραφές, γραμμένο σε PHP. Αποτελεί ένα ιδανικό εργαλείο για την ανάπτυξη μικρών έως μεγάλων δυναμικών κοινοτικών ιστοσελίδων, εταιρικών πυλώνων, ιστολογίων και πολλών άλλων.

Το XOOPS διανέμεται υπό τους όρους της GNU General Public License (GPL) έκδοση 2 ή νεότερη, και είναι ελεύθερο για χρήση και τροποποίηση. Είναι ελεύθερο για αναδιανομή εφόσον τηρείτε τους όρους διανομής της GPL.

Απαιτήσεις

  • Διακομιστής WWW (Apache, NGINX, IIS, κ.λπ.)
  • PHP 7.4 ή νεότερη, συνιστάται 8.3.0+
  • MySQL 5.7 ή νεότερη, συνιστάται MySQL 8.4.5+

Πριν την εγκατάσταση

  1. Ρυθμίστε σωστά τον διακομιστή WWW, την PHP και τον διακομιστή βάσης δεδομένων.
  2. Ετοιμάστε μια βάση δεδομένων για τον ιστότοπό σας XOOPS.
  3. Ετοιμάστε έναν λογαριασμό χρήστη και χορηγήστε του πρόσβαση στη βάση δεδομένων.
  4. Κάντε τους καταλόγους και τα αρχεία εγγράψιμα: %s
  5. Για λόγους ασφαλείας, συνιστάται ιδιαίτερα να μετακινήσετε τους δύο παρακάτω καταλόγους εκτός του root εγγράφων και να αλλάξετε τα ονόματα των φακέλων: %s
  6. Δημιουργήστε (αν δεν υπάρχουν ήδη) και κάντε τους καταλόγους εγγράψιμους: %s
  7. Ενεργοποιήστε τα cookies και την JavaScript του προγράμματος περιήγησής σας.

Εγκατάσταση τοπικά

Εάν χρησιμοποιείτε ένα τοπικό περιβάλλον για ανάπτυξη ή δοκιμή, βεβαιωθείτε ότι πληρούνται οι προηγούμενες απαιτήσεις. Μόλις γίνει αυτό, αντιγράψτε τα περιεχόμενα του καταλόγου htdocs (από το αρχείο διανομής του XOOPS) στην κύρια διαδρομή εγγράφων του διαδικτυακού σας περιβάλλοντος. Μόλις αντιγραφούν τα αρχεία εκεί, μπορείτε να ξεκινήσετε την εγκατάσταση κατευθύνοντας το πρόγραμμα περιήγησής σας στο "http://yoursite.com". Αυτό θα ξεκινήσει τη διαδικασία εγκατάστασης.

Εγκατάσταση σε φιλοξενούμενη πλατφόρμα

Εάν χρησιμοποιείτε ένα φιλοξενούμενο περιβάλλον, αποσυμπιέστε τα αρχεία του XOOPS τοπικά ή στον διακομιστή εάν έχετε πρόσβαση σε κέλυφος (δηλ. SSH). Μόλις γίνει αυτό, φροντίστε να μετακινήσετε ή να αντιγράψετε όλα τα αρχεία του XOOPS από τον κατάλογο htdocs στον κύριο διαδικτυακό κατάλογό σας (ο πάροχός σας συνήθως παρέχει αυτήν την τοποθεσία με οδηγίες). Μόλις αντιγραφούν τα αρχεία εκεί, μπορείτε να ξεκινήσετε την εγκατάσταση κατευθύνοντας το πρόγραμμα περιήγησής σας στο "http://yoursite.com". Αυτό θα ξεκινήσει τη διαδικασία εγκατάστασης.
