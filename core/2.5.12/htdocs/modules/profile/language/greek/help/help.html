Βοήθεια: Προφίλ

ΠΕΡΙΓΡΑΦΗ

Η ενότητα Προφίλ είναι για τη διαχείριση προσαρμοσμένων πεδίων προφίλ χρήστη.

ΕΓΚΑΤΑΣΤΑΣΗ/ΑΠΕΓΚΑΤΑΣΤΑΣΗ

Δεν απαιτούνται ειδικές ενέργειες, ακολουθήστε την τυπική διαδικασία εγκατάστασης – εξαγάγετε τον φάκελο /profile στον κατάλογο ../modules. Εγκαταστήστε την ενότητα μέσω Διαχείριση -> Ενότητα Συστήματος -> Ενότητες.

Λεπτομερείς οδηγίες για την εγκατάσταση ενοτήτων είναι διαθέσιμες στο Εγχειρίδιο Λειτουργίας XOOPS

ΟΔΗΓΙΕΣ ΛΕΙΤΟΥΡΓΙΑΣ

Για να ρυθμίσετε αυτή την ενότητα, χρειάζεται να:

  • Ρυθμίσετε τις προτιμήσεις σας για την ενότητα (δείτε ‘Προτιμήσεις’) και προαιρετικά το μπλοκ Προφίλ αν σκοπεύετε να το χρησιμοποιήσετε (δείτε ‘Μπλοκ’).
  • Επεξεργαστείτε τις υπάρχουσες Κατηγορίες ή προσθέστε νέες.
  • Επεξεργαστείτε τα υπάρχοντα Πεδία ή προσθέστε νέα. Εδώ μπορείτε να καθορίσετε ποια πεδία θα είναι ορατά σε ποια κατηγορία, και αν θα είναι ορατά κατά την εγγραφή χρήστη.
  • Καθορίστε τη σειρά των βημάτων Εγγραφής.
  • Και τέλος, μπορείτε να ορίσετε δικαιώματα για μεμονωμένα πεδία - ποια είναι επεξεργάσιμα, ποια είναι αναζητήσιμα.

Μέτρα κατά του Spam

Για να ελαχιστοποιήσετε τις εγγραφές spam, κάντε τα εξής:

  • μεταβείτε στην ενότητα Protector στην Διαχείριση, πηγαίνετε στις Προτιμήσεις, και στη συνέχεια στο κάτω μέρος, σε αυτή την επιλογή: "Stop Forum Spam" Ελέγχει τα δεδομένα POST έναντι spammers που είναι εγγεγραμμένοι στη βάση δεδομένων www.stopforumspam.com.
    Ορίστε το σε "Αποκλεισμός της IP (χωρίς όριο)"

  • στο /class/captcha/config.php, βεβαιωθείτε ότι η λειτουργία έχει οριστεί σε "text":

    return $config = array(
    'disabled' => false, // Απενεργοποίηση του CAPTCHA
    'mode' => 'text', // προεπιλεγμένη λειτουργία, μπορείτε να επιλέξετε 'text', 'image', 'recaptcha'(απαιτεί κλειδί api)
    'name' => 'xoopscaptcha', // όνομα captcha
    'skipmember' => true, // Παράλειψη ελέγχου CAPTCHA για μέλη
    'maxattempts' => 10, // Μέγιστες προσπάθειες για κάθε συνεδρία
    );

  • Στην ενότητα Προφίλ, μεταβείτε στην Διαχείριση και στο Βασικό βήμα που βρίσκεται στο:
    /modules/profile/admin/step.php?id=1
    ορίστε το "Αποθήκευση μετά τη διακοπή" σε "Όχι"


  • Στις Προτιμήσεις Προφίλ, ορίστε την επιλογή "Χρήση Captcha μετά το δεύτερο βήμα εγγραφής" σε "Ναι" (είναι η προεπιλογή)
