Βοήθεια: Banners

Περιγραφή

Τα banners είναι διαφημίσεις που εμφανίζονται στο επάνω τμήμα της σελίδας. Μπορεί να είναι εικόνες ή ταινίες flash, όπως το έντονα περιστρεφόμενο banner του XOOPS που αποτελεί μέρος της προεπιλεγμένης εγκατάστασης. Το XOOPS διαθέτει ένα ενσωματωμένο σύστημα για τη διαχείριση banners και τη δημιουργία λογαριασμών για διαφημιστικούς πελάτες, το οποίο βρίσκεται στην Ενότητα Συστήματος -> Banners. Μπορείτε να έχετε πολλαπλά banners σε έναν ιστότοπο XOOPS – η περιοχή εμφάνισης εναλλάσσεται διαδοχικά μεταξύ κάθε ενεργού banner κάθε φορά που φορτώνεται μια νέα σελίδα.

Κάθε φορά που ένα banner εμφανίζεται σε έναν επισκέπτη, αυτό μετράει ως μία ‘εμφάνιση’, η οποία είναι το κοινό εμπορικό νόμισμα των banners. Οι πελάτες συνήθως πληρώνουν για έναν συγκεκριμένο αριθμό εμφανίσεων – δηλαδή για να εμφανιστεί η διαφήμισή τους έναν συγκεκριμένο αριθμό φορών.

Σημείωση: Αν ασχολείστε με την πώληση banners με βάση τον αριθμό των εμφανίσεων, υπάρχει ένα σημαντικό πράγμα που πρέπει να κάνετε: Μεταβείτε στην Ενότητα Συστήματος -> Προτιμήσεις -> Γενικές Ρυθμίσεις και συμπληρώστε το πεδίο ‘Η Διεύθυνση IP Σας’ (αν έχετε). Αυτό θα αποτρέψει την συμπερίληψη των επισκέψεών σας στον ιστότοπο στον αριθμό των εμφανίσεων. Η παράλειψη αυτού θα μπορούσε να θεωρηθεί κάπως ανέντιμη (σημειώνοντας, ωστόσο, ότι πολλοί Διαχειριστές δεν έχουν μόνιμη IP – π.χ. όσοι χρησιμοποιούν dial up modem).
