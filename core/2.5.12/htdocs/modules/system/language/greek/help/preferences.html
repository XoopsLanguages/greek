Βοήθεια: Προτιμήσεις

Περιγραφή

Ο πίνακας προτιμήσεων είναι όπου ρυθμίζετε τις γενικές προτιμήσεις του ιστότοπού σας. Αξίζει να αφιερώσετε χρόνο για να τις επεξεργαστείτε, καθώς κάποιες ρυθμίσεις μπορεί να έχουν σημαντική επίδραση στην απόδοση, την ασφάλεια και την ευχρηστία του ιστότοπού σας.

Γενικές Ρυθμίσεις

Αυτές είναι οι πιο σημαντικές και οι πιο συχνά προσβάσιμες ρυθμίσεις. Θα ρυθμίσετε το όνομα του ιστότοπου και το σύνθημα για την ιστοσελίδα σας. Θα επιλέξετε επίσης το προεπιλεγμένο θέμα και το GUI διαχειριστή. Αν χρειάζεστε να εργαστείτε στον ιστότοπό σας, μπορείτε να τον κλείσετε για όλους τους χρήστες. Μπορείτε επίσης να ορίσετε τους χρόνους προσωρινής μνήμης σε επίπεδο ενότητας.

Meta Tags και Υποσέλιδο

Τα meta tags παρέχουν πληροφορίες σχετικά με τον ιστότοπό σας – περιγράφουν τη φύση του ιστότοπου. Η κύρια χρήση τους είναι να βοηθήσουν τις μηχανές αναζήτησης και τα ρομπότ τους να ευρετηριάσουν σωστά τον ιστότοπό σας, δηλαδή να βοηθήσουν τους σωστούς χρήστες να τον βρουν. Οι πληροφορίες που παρέχετε περιλαμβάνονται στην κεφαλίδα κάθε σελίδας του ιστότοπου. Θα πρέπει να αντιπροσωπεύουν το περιεχόμενο του ιστότοπου ως σύνολο – οι meta πληροφορίες είναι οι ίδιες σε κάθε σελίδα και αυτή τη στιγμή δεν μπορούν να εξατομικευτούν για διαφορετικές ενότητες ή σελίδες.

Να είστε προσεκτικοί καθώς οι μεγάλες μηχανές αναζήτησης χρησιμοποιούν διαφορετικές στρατηγικές για την ευρετηρίαση ιστοσελίδων. Μπορεί να δώσουν διαφορετική βαρύτητα στα meta tags ή να τα αγνοήσουν εντελώς. Η προσεκτική χρήση των meta πληροφοριών μπορεί να βοηθήσει τον ιστότοπό σας να κατατάσσεται καλύτερα στις μηχανές αναζήτησης και είναι χρήσιμο να αναζητήσετε πληροφορίες για τις στρατηγικές ευρετηρίασης των κυριότερων παραπομπών σας (δείτε το αρχείο καταγραφής του ιστότοπού σας). Αυτό μπορεί να σας βοηθήσει να βελτιστοποιήσετε τις meta πληροφορίες ή άλλες πτυχές του σχεδιασμού και του περιεχομένου του ιστότοπού σας για να βελτιώσετε τις κατατάξεις σας.

Προσπαθήστε να είστε ακριβείς και ειλικρινείς όταν παρέχετε meta πληροφορίες. Μην προσπαθήσετε να «μανιπουλάρετε» τις μηχανές αναζήτησης για να αποκτήσετε καλύτερη κατάταξη – μπορεί να αναγνωρίσουν αυτή τη συμπεριφορά και ο ιστότοπός σας να υποβαθμιστεί ή να αποκλειστεί από το ευρετήριο τους.

Τα πεδία Meta Copyright και Υποσέλιδο υποστηρίζουν και τις δύο ετικέτες αντικατάστασης για να διευκολύνουν τη συντήρηση.
 - {X_SITEURL} θα αντικατασταθεί με τη πλήρη διεύθυνση URL της κύριας σελίδας του ιστότοπού σας.
 - {X_YEAR} θα αντικατασταθεί με το τρέχον έτος (τέσσερα ψηφία). Αυτό εξαλείφει την ανάγκη να ενημερώνετε την ημερομηνία πνευματικών δικαιωμάτων κάθε χρόνο.

Επιλογές Λογοκρισίας Λέξεων

Το XOOPS μπορεί αυτόματα να λογοκρίνει ανεπιθύμητες λέξεις που μπορεί να δημοσιεύσουν οι επισκέπτες στον ιστότοπό σας και να τις αντικαταστήσει με έναν γενικό, εναλλακτικό όρο της επιλογής σας. Η λειτουργία λογοκρισίας καταναλώνει πόρους του συστήματος και μπορεί να επιβραδύνει κάπως τον ιστότοπό σας. Ενεργοποιείται από προεπιλογή.

Για να ενεργοποιήσετε τη λογοκρισία στον ιστότοπό σας, επιλέξτε το κουμπί ‘enable’. Πρέπει να εισαγάγετε μια λίστα με τις ανεπιθύμητες λέξεις στο πλαίσιο ‘words to censor’, χωρίζοντας τις με τον χαρακτήρα σωλήνα. Αυτή η λειτουργία δεν διαφοροποιεί τα κεφαλαία από τα πεζά. Εισαγάγετε έναν κατάλληλο όρο αντικατάστασης στο πλαίσιο ‘replace bad words with’. Η προεπιλογή είναι #OOPS#.

Επιλογές Αναζήτησης

Η λειτουργία αναζήτησης είναι ένα από τα πιο σημαντικά εργαλεία για την εύρεση πληροφοριών σε έναν ιστότοπο πύλης. Η αναζήτηση στο XOOPS κατατάσσει τα αποτελέσματα κατά ενότητα – τα αρχεία αναγνωρίζονται ως άρθρα ειδήσεων, λήψεις, συνδέσμους ή άλλους τύπους σύμφωνα με τις ενότητες που έχετε εγκαταστήσει.

Οι παγκόσμιες αναζητήσεις σε όλους τους πόρους του ιστότοπού σας είναι ενεργοποιημένες από προεπιλογή. Εάν απενεργοποιήσετε τις παγκόσμιες αναζητήσεις, αυτή η λειτουργία θα είναι πλέον μη διαθέσιμη και το μπλοκ αναζήτησης θα καταστεί μη λειτουργικό. Το ελάχιστο μήκος λέξης για τις αναζητήσεις είναι πέντε χαρακτήρες. Οποιεσδήποτε λέξεις είναι μικρότερες θα αγνοούνται. Αυτό είναι χρήσιμο για τον έλεγχο της ακρίβειας των αναζητήσεων, αν και δεν είναι πολύ χρήσιμο για όρους όπως «χάρτης ιστότοπου». Οι μικρές λέξεις-κλειδιά είναι πιο κοινές, επομένως συνήθως επιστρέφουν περισσότερα αποτελέσματα, πολλά από τα οποία μπορεί να είναι άσχετα. Οι μεγαλύτερες λέξεις-κλειδιά συνήθως επιστρέφουν λιγότερα αποτελέσματα αλλά με υψηλότερη ακρίβεια.

Η αναζήτηση είναι σχετικά ακριβή σε υπολογιστικούς όρους, οπότε ο αποκλεισμός μικρών όρων θα μειώσει το φόρτο στον διακομιστή. Ωστόσο, να θυμάστε ότι μερικές φορές είναι χρήσιμο να μπορείτε να αναζητάτε για έναν μικρό όρο.

Ρυθμίσεις Email

Αυτό το τμήμα του πίνακα ελέγχου χρησιμοποιείται για τη διαμόρφωση του email για το σύστημα. Συνιστάται να αφήσετε τη μέθοδο παράδοσης του email στην προεπιλεγμένη ρύθμιση «PHP (mail)». Δεν χρειάζεται να συμπληρώσετε τα πεδία Sendmail ή SMTP εκτός αν χρησιμοποιείτε μια εναλλακτική μέθοδο παράδοσης email.

Επιλογές Επαλήθευσης

Αφήστε το στην προεπιλεγμένη ρύθμιση της βάσης δεδομένων XOOPS και μπορείτε να αγνοήσετε όλες τις άλλες επιλογές εκεί.

Οι εξυπηρετητές LDAP ή Microsoft Active Directory έχουν προστεθεί για επαλήθευση ως εναλλακτική λύση στους κανονικούς χρήστες του XOOPS. Ο λόγος είναι ότι σε πολλές εταιρείες υπάρχει ήδη μια βάση δεδομένων χρηστών σε ADS ή LDAP, και αυτοί μπορούν να τη χρησιμοποιήσουν για να συνδεθούν στο XOOPS χωρίς να χρειάζεται να εγγραφούν με έναν ακόμη λογαριασμό.

Ρυθμίσεις Συστημικής Ενότητας

Εδώ μπορείτε να ενεργοποιήσετε/απενεργοποιήσετε κάθε τμήμα του Συστήματος, καθώς και να καθορίσετε τις προεπιλεγμένες ρυθμίσεις για το σύστημα, π.χ. για τον αριθμό των εμφανιζόμενων αβατάρ ή εικόνων ανά σελίδα. Μπορείτε επίσης να επιλέξετε έναν προεπιλεγμένο επεξεργαστή για τα μπλοκ, τα σχόλια και για όλες τις ενότητες.
