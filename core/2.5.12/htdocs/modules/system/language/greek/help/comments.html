Βοήθεια: Σχόλια

Περιγραφή

Οι διαχειριστές του ιστότοπου έχουν την επιλογή να παρέχουν μια ενότητα σχολίων για τους χρήστες. Σχόλια μπορούν να προστεθούν σε πολλές ενότητες και να εμφανιστούν με διάφορους τρόπους. Τα σχόλια μπορούν να ελεγχθούν, να φιλτραριστούν και να είναι διαθέσιμα σε επιλεγμένες ομάδες. Αυτό παρέχει γρήγορη και εύκολη αλληλεπίδραση από τους χρήστες του ιστότοπου, ενώ διατηρείται καλή εποπτεία και ασφάλεια.

Ο Διαχειριστής Σχολίων επιτρέπει στον διαχειριστή του ιστότοπου να επεξεργαστεί και να διαγράψει οποιοδήποτε σχόλιο έχει δημοσιευτεί στον ιστότοπο, προερχόμενο από οποιαδήποτε ενότητα υποστηρίζει τη λειτουργία σχολίων. Όπως και η σελίδα Διαχείρισης Μπλοκ, κάποιες επιλογές φίλτρων είναι διαθέσιμες για να βοηθήσουν τον διαχειριστή του ιστότοπου να διαχειριστεί τα σχόλια.

Μπορείτε επίσης να επιλέξετε πολλά σχόλια και να τα διαγράψετε ταυτόχρονα.
