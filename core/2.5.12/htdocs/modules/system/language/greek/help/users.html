Βοήθεια: Χρήστες

Περιγραφή

Το XOOPS παρέχει ενσωματωμένα εργαλεία για εύκολη διαχείριση χρηστών. Αυτά περιλαμβάνουν τη δυνατότητα αναζήτησης χρηστών με διάφορα κριτήρια, αποστολή email και ιδιωτικών μηνυμάτων στους χρήστες μέσω ενός συστήματος μηνυμάτων που βασίζεται σε πρότυπα. Στον ιστότοπό σας, οι χρήστες έχουν τα δικαιώματα που αντιστοιχούν στην ομάδα στην οποία έχουν ανατεθεί.

Όσον αφορά τους χρήστες, έχουν πρόσβαση στο προφίλ τους, έχουν κάποια δυνατότητα ελέγχου στις περισσότερες πληροφορίες που θα είναι ορατές σε άλλους χρήστες, και διαθέτουν επίσης έναν εισερχόμενο φάκελο για να λαμβάνουν ιδιωτικά μηνύματα από τον διαχειριστή του ιστότοπου ή από άλλους χρήστες. Επίσης, έχουν έλεγχο σε ποια αλλαγές στον ιστότοπο θα ενεργοποιήσουν προσωπικές ειδοποιήσεις. Και ανάλογα με τα δικαιώματα που έχουν οριστεί από τον διαχειριστή του ιστότοπου, μπορούν να δουν περιεχόμενο και να συμμετέχουν στη ζωή του ιστότοπου ανεβάζοντας περιεχόμενο, γράφοντας ιστορίες ή σχόλια, αξιολογώντας περιεχόμενο και γενικά δίνοντας την ανατροφοδότησή τους για ό,τι βλέπουν στον ιστότοπο.

Εξελιγμένη Αναζήτηση

Η "Εξελιγμένη Αναζήτηση" παρέχει μια ευρεία γκάμα επιλογών που μπορείτε να χρησιμοποιήσετε για να αναζητήσετε συγκεκριμένους χρήστες ή να φιλτράρετε τα μέλη σας. Μπορείτε επίσης να αποκτήσετε μια πλήρη λίστα μελών πατώντας το κουμπί ‘υποβολή’ χωρίς να χρησιμοποιήσετε κανένα από τα φίλτρα.

Επεξεργασία Χρηστών

Μπορείτε να προσθέσετε, να επεξεργαστείτε και να διαγράψετε χρήστες μέσω αυτού του τμήματος της Ενότητας Συστήματος.

Για να επεξεργαστείτε έναν χρήστη, αναζητήστε τον βάσει του ονόματος ή άλλων κριτηρίων. Στη συνέχεια, μπορείτε να επιλέξετε είτε

i) να τροποποιήσετε τα στοιχεία τους ή
ii) να τους διαγράψετε χρησιμοποιώντας το δεύτερο αναπτυσσόμενο μενού.

Επιλέγοντας ‘τροποποίηση’ θα ανοίξει το προφίλ του χρήστη το οποίο μπορείτε να αλλάξετε όπως επιθυμείτε (σημειώστε ότι μπορείτε επίσης να φτάσετε στη σελίδα επεξεργασίας προφίλ από την πλευρά του χρήστη μέσω του καταλόγου μελών ή κάνοντας κλικ σε ένα όνομα χρήστη οπουδήποτε στον ιστότοπο).
