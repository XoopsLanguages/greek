Βοήθεια: Πρότυπα

Περιγραφή

Το XOOPS επιτρέπει κάποιον διαχωρισμό μεταξύ της πραγματικής παρουσίασης του ιστότοπου (το “skin” ή διεπαφή χρήστη) και του κώδικα που εκτελείται στο παρασκήνιο. Το πρώτο διαχειρίζεται ένας εξελιγμένος μηχανισμός που βασίζεται σε γραφικά θέματα. Ένα θέμα μπορεί να περιλαμβάνει, στη βασική του μορφή, τα απαραίτητα πρότυπα HTML που ορίζουν την οπτική διεπαφή του ιστότοπου, καθώς και τα φύλλα στυλ και τις εικόνες που απαιτούνται για να συμπληρώσουν αυτή την οπτική διεπαφή. Τα θέματα είναι εναλλάξιμα στη διαχειριστική διεπαφή, επομένως η αλλαγή ολόκληρης της εμφάνισης ενός ιστότοπου XOOPS είναι απλώς θέμα αλλαγής της τιμής ενός επιλογέα και κλικ στο “Αποδοχή”. Το XOOPS παρέχει επίσης στους διαχειριστές ιστότοπων τη δυνατότητα να επιτρέπουν στους χρήστες να επιλέγουν ανάμεσα σε κάποια διαθέσιμα θέματα.

Γενικά μιλώντας, ένα θέμα είναι το οπτικό πλαίσιο υψηλού επιπέδου ενός ιστότοπου: το θέμα ορίζει, για παράδειγμα, τη γενική διάταξη του ιστότοπου. από την άλλη πλευρά, συνήθως δεν ορίζει τη συγκεκριμένη κατανομή περιεχομένου μέσα σε ένα μπλοκ. Αυτό διαχειρίζεται μια διαφορετική οντότητα που ονομάζεται σύνολο προτύπων. Ένα σύνολο προτύπων είναι ένα σύνολο προτύπων HTML που ορίζουν την παρουσίαση περιεχομένου σε όλα τα μπλοκ που αποτελούν έναν ιστότοπο XOOPS. Αν θέλουμε μια πιο σαφή μεταφορά για αυτό, θα μπορούσαμε να πούμε ότι το θέμα είναι σαν τον αρχιτέκτονα ενός σπιτιού, ενώ το σύνολο προτύπων είναι σαν τον διακοσμητή εσωτερικών χώρων. Ο πρώτος θέτει τους τοίχους και ορίζει τη γενική κατανομή, αλλά ο δεύτερος είναι υπεύθυνος για το “ντύσιμο” αυτών των τοίχων.

Εάν το ενεργό θέμα είναι ένα απλό θέμα που περιλαμβάνει μόνο ένα αρχείο θέματος, αρχεία CSS και κάποιες εικόνες, το XOOPS χρησιμοποιεί το προεπιλεγμένο σύνολο προτύπων για να ορίσει την παρουσίαση των μπλοκ. Ωστόσο, ορισμένα σύνθετα θέματα είναι αρκετά εξελιγμένα ώστε να περιλαμβάνουν το δικό τους σύνολο προτύπων. Σε αυτή την περίπτωση, ο διαχειριστής του ιστότοπου πρέπει να χρησιμοποιήσει τη διαχειριστική διεπαφή για να ορίσει τόσο το θέμα όσο και το σύνολο προτύπων.

Πώς επιτυγχάνεται η μαγεία των θεμάτων; Δηλαδή, πώς μπορούν τα πρότυπα HTML να περιλαμβάνουν δυναμικό περιεχόμενο; Αυτό δεν γίνεται με καθρέφτες και καπνό, αλλά με μια έξυπνη μηχανή προτύπων που ονομάζεται Smarty, η οποία κάνει τη λειτουργία αυτού που παλαιότερα ονομαζόταν παρεμβολή μεταβλητών. Εκφραζόμενο απλά, θα μπορούσαμε να πούμε ότι το σύστημα XOOPS χρησιμοποιεί αρχεία PHP για να διατηρήσει τη λογική των ενοτήτων (πρόσβαση στη βάση δεδομένων, εκχώρηση μεταβλητών, διαχείριση δικαιωμάτων...), αρχεία HTML με μεταβλητές Smarty εισαγμένες για τη διαχείριση της πραγματικής οπτικής διεπαφής και τη μηχανή Smarty για την εισαγωγή τιμών και συμβολοσειρών σε συγκεκριμένα σημεία μέσα στην HTML που δημιουργεί τις σελίδες που βλέπει ο τελικός χρήστης. Συνολικά, αυτό που σημαίνει για εσάς μπορεί εύκολα να συνοψιστεί: χάρη στο σύστημα θεμάτων του, το XOOPS επιτρέπει στους σχεδιαστές ιστότοπων να δημιουργούν πιο ενδιαφέροντες και οπτικά εντυπωσιακούς ιστότοπους χωρίς μεγάλη ανησυχία για το τι συμβαίνει στο παρασκήνιο.

Κλωνοποίηση και λήψη συνόλων προτύπων

Αν θέλετε να δημιουργήσετε ένα προσαρμοσμένο σύνολο προτύπων, μπορείτε να δημιουργήσετε ένα αντίγραφο του προεπιλεγμένου συνόλου (ή οποιουδήποτε άλλου) επιλέγοντας τον σύνδεσμο ‘κλωνοποίηση’ (θα σας ζητηθεί να δώσετε ένα όνομα στο νέο σύνολο). Σε αντίθεση με το προεπιλεγμένο, άλλα σύνολα προτύπων μπορούν να επεξεργαστούν ελεύθερα. Μπορείτε επίσης να αποθηκεύσετε ένα αντίγραφο ενός συνόλου προτύπων στον δίσκο επιλέγοντας τον σύνδεσμο ‘λήψη’ ή να το καταργήσετε με τον σύνδεσμο ‘διαγραφή’. Μπορεί επίσης μερικές φορές να δείτε έναν επιπλέον σύνδεσμο ‘Δημιουργία’ δίπλα σε νέες ενότητες για τις οποίες δεν έχουν δημιουργηθεί ακόμη πρότυπα.

Επεξεργασία προτύπων

Μπορείτε να δείτε μια πλήρη λίστα των διαθέσιμων προτύπων περιηγούμενοι στο δενδροειδές διάγραμμα καταλόγων. Όταν επιλέξετε ένα πρότυπο, το XOOPS θα το ανοίξει σε έναν επεξεργαστή όπου μπορείτε να το επεξεργαστείτε.

Κατά την επεξεργασία προτύπων, πρέπει να ενεργοποιήσετε μια προτίμηση που ενημερώνει αυτόματα τα αρχεία προτύπων .tpl της ενότητας εάν υπάρχουν νεότερα αρχεία στον κατάλογο themes/το θέμα σας/templates για το τρέχον θέμα (Ενότητα Συστήματος -> Προτιμήσεις -> Γενικές Ρυθμίσεις -> Ενημέρωση αρχείων προτύπων .tpl ενότητας από τον κατάλογο themes/το θέμα σας/templates; -> ‘Ναι’). Αυτό θα πρέπει να απενεργοποιηθεί μόλις ο ιστότοπος γίνει δημόσιος.

Αν κατά λάθος καταστρέψετε ένα πρότυπο, μην ξεχνάτε ότι μπορείτε να ανακτήσετε ένα ‘καλό’ αντίγραφο από το προεπιλεγμένο σύνολο προτύπων. Αν εργάζεστε με μη τυπικά πρότυπα, ίσως είναι καλή ιδέα να αποθηκεύσετε ένα αντίγραφο στον δίσκο πριν αρχίσετε να τα τροποποιείτε.
