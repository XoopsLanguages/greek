Βοήθεια: Smilies

Περιγραφή

Τα Smilies είναι μικρά γραφικά σύμβολα που οι χρήστες μπορούν να χρησιμοποιήσουν για να εκφραστούν. Μπορούν να εισαχθούν οπουδήποτε στο κείμενο ενός άρθρου, σχολίου ή δημοσίευσης τόσο από την πλευρά του χρήστη όσο και από την πλευρά του διαχειριστή του συστήματος. Το XOOPS περιλαμβάνει μια συλλογή 17 ενσωματωμένων smilies. Μερικά από αυτά εμφανίζονται στις φόρμες υποβολής σε διάφορες ενότητες (συμπεριλαμβανομένων των φόρουμ, των ειδήσεων, των λήψεων, των συνδέσμων και άλλων). Κάνοντας κλικ σε ένα smilie θα το εισαγάγετε στη δημοσίευση. Τα Smilies μπορούν επίσης να εισαχθούν πληκτρολογώντας τους μοναδικούς τους ‘κωδικούς’ (συντομεύσεις πληκτρολογίου), για παράδειγμα, πληκτρολογώντας ':' '-' 'D' σε μια δημοσίευση θα εισαγάγετε αυτό το smilie:

Η πλήρης λίστα των smilies που είναι διαθέσιμα στο σύστημά σας και οι κωδικοί τους είναι προσβάσιμοι μέσω των Smilies, όπου βρίσκεστε τώρα. Εδώ μπορείτε να προσθέσετε, να επεξεργαστείτε και να διαγράψετε smilies, και να ελέγξετε ποια εμφανίζονται στις φόρμες υποβολής (μόνο εννέα από τα 17 εμφανίζονται από προεπιλογή)

Αν δεν θέλετε smilies στο σύστημά σας, μπορείτε είτε να καταργήσετε την επιλογή από το πλαίσιο "εμφάνιση στη φόρμα;" (οπότε δεν θα εμφανίζονται στις φόρμες υποβολής, αλλά θα εξακολουθούν να είναι διαθέσιμα μέσω των κωδικών τους), είτε να χρησιμοποιήσετε τον σύνδεσμο διαγραφής για να τα καταργήσετε εντελώς.
