Βοήθεια: Ομάδες

Περιγραφή

Το XOOPS ενσωματώνει ένα σύστημα εγγραφής μελών - μπορείτε προαιρετικά να απαιτήσετε από τους χρήστες να εγγραφούν στον ιστότοπό σας για να έχουν πρόσβαση σε συγκεκριμένες περιοχές ή λειτουργίες. Για παράδειγμα, οι περισσότεροι ιστότοποι απαιτούν από τους χρήστες να εγγραφούν πριν τους επιτραπεί να συνεισφέρουν ειδήσεις ή να δημοσιεύουν μηνύματα στα φόρουμ.
Τα δικαιώματα πρόσβασης και διαχείρισης σε έναν ιστότοπο XOOPS ορίζονται μέσω ενός ευέλικτου συστήματος αδειών που βασίζεται σε «ομάδες χρηστών». Οι προεπιλεγμένες ομάδες περιλαμβάνουν ανώνυμους χρήστες, εγγεγραμμένους χρήστες και διαχειριστές ιστοσελίδων, αλλά φυσικά μπορείτε να ορίσετε όσες ομάδες χρειάζεστε για τη διαχείριση του ιστότοπού σας. Κάθε ομάδα μπορεί να λάβει ένα μοναδικό σύνολο δικαιωμάτων πρόσβασης που καθορίζουν:

- Ποια μπλοκ μπορούν να εμφανιστούν
- Ποια modules μπορούν να προσπελαστούν
- Ποια modules μπορούν να διαχειριστούν
- Ποιες πτυχές της διαχείρισης του συστήματος μπορούν να τροποποιηθούν

Μπορείτε να αλλάξετε το όνομα αυτών των προεπιλεγμένων ομάδων, αλλά δεν μπορείτε να τις διαγράψετε. Άλλες ομάδες χρηστών μπορούν επίσης να οριστούν και να καθοριστούν όπως συζητείται παρακάτω.

Προσαρμοσμένες ομάδες μπορούν επίσης να δημιουργηθούν μέσω της σελίδας διαχείρισης ομάδων.
