Βοήθεια: Μπλοκ

Περιγραφή

Τα μπλοκ είναι διακριτά τμήματα περιεχομένου που μπορούν να δημιουργηθούν και να ρυθμιστούν στη διεπαφή διαχείρισης. Μπορούν να δημιουργηθούν προσαρμοσμένα μπλοκ, τα οποία συνήθως αποτελούνται από κείμενο, γραφικά και εικόνες. Το περιεχόμενο αυτών των μπλοκ μπορεί να μορφοποιηθεί ξεχωριστά ή να κληρονομήσει τη μορφοποίηση του ιστότοπου. Τα μπλοκ μπορούν επίσης να αποτελούν μέρος της εμφάνισης μιας ενότητας. Η σελίδα διαχείρισης μπλοκ είναι όπου ρυθμίζονται και διαχειρίζονται όλα τα μπλοκ.

Τα μπλοκ είναι σαν την πρώτη σελίδα μιας εφημερίδας: είναι σύντομες αναφορές που παροτρύνουν τους αναγνώστες να διαβάσουν πιο ολοκληρωμένες ιστορίες στις εσωτερικές σελίδες. Τα μπλοκ προσφέρουν μικρά τμήματα δυναμικού περιεχομένου με συνδέσμους προς το πραγματικό περιεχόμενο που είναι αποθηκευμένο στην ενότητα.

Μέρος των πραγμάτων που συμβαίνουν στο παρασκήνιο ή πίσω από τις κουρτίνες όταν εγκαθίσταται μια ενότητα σε έναν ιστότοπο XOOPS είναι ότι τα μπλοκ της ενότητας καταχωρούνται στο σύστημα, ώστε να μπορούν να χρησιμοποιηθούν κατά βούληση από τον διαχειριστή του ιστότοπου.

Τι μπορεί να κάνει ένας διαχειριστής ιστότοπου με ένα μπλοκ; Λοιπόν, κυρίως, να αποφασίσει πού στον ιστότοπο θα εμφανίζεται το μπλοκ (το ισοδύναμο του καθορισμού μιας βιτρίνας καταστήματος, η οποία σε αυτή την περίπτωση μπορεί να είναι εξωτερική, η αρχική σελίδα ή ακόμα και εσωτερική: ένα μπλοκ μπορεί να οριστεί να εμφανίζεται σε άλλες ενότητες) και, όταν εμφανίζεται, πού μέσα σε μια σελίδα θα είναι ορατό.

Κάθε ενότητα που είναι εγκατεστημένη σε έναν ιστότοπο XOOPS καταχωρεί τα μπλοκ της στο σύστημα, και ο διαχειριστής του ιστότοπου μπορεί να αποφασίσει πού θα εμφανίζεται, σε ποιο μέρος της σελίδας θα εμφανίζεται και τι είδους συνάφεια, βάρος ή θέση θα έχει. Ο διαχειριστής του ιστότοπου μπορεί επίσης να επεξεργαστεί τον τίτλο του μπλοκ και, αν έχει αρκετές γνώσεις, μπορεί ακόμη και να επεξεργαστεί το πρότυπο που βρίσκεται πίσω από την παρουσίαση του μπλοκ.

Συμβουλές

Μπορείτε να αποκρύψετε τον τίτλο που θα εμφανίζεται όταν το μπλοκ προβάλλεται προσθέτοντας "// " (δηλαδή κάθετος-κάθετος-κενό) ως τους τρεις πρώτους χαρακτήρες του τίτλου. Με αυτόν τον τρόπο μπορείτε να συνδυάσετε απρόσκοπτα πολλά μπλοκ για να δημιουργήσετε την προβολή που θέλετε.

Μπορείτε να δημιουργήσετε τα δικά σας μπλοκ περιεχομένου χρησιμοποιώντας το κουμπί "Προσθήκη Μπλοκ". Μπορείτε να δημιουργήσετε το δικό σας περιεχόμενο σε HTML, απλό κείμενο με Κώδικες XOOPS ή ακόμα και PHP. Στα προσαρμοσμένα μπλοκ σας, ο κώδικας "{X_SITEURL}" θα αντικατασταθεί με τη διεύθυνση URL του ιστότοπού σας με μια τελική κάθετο.
