Βοήθεια: Αποστολή Email σε Χρήστες

Περιγραφή

Το XOOPS διαθέτει ένα ενσωματωμένο σύστημα ανταλλαγής μηνυμάτων που σας επιτρέπει να στείλετε ένα μήνυμα σε ολόκληρη τη βάση των εγγεγραμμένων χρηστών σας ή σε υποομάδες αυτών, ή χρησιμοποιώντας φίλτρα ημερομηνίας. Αυτό μπορεί να είναι είτε με τη μορφή ιδιωτικού μηνύματος (δηλαδή μέσω του ενσωματωμένου συστήματος ανταλλαγής μηνυμάτων του XOOPS) είτε μέσω email. Διατίθεται μια σειρά επιλογών για το φιλτράρισμα υποομάδων χρηστών από τη βάση μελών σας.

Μπορείτε επίσης να χρησιμοποιήσετε προκαθορισμένες ετικέτες που θα σας επιτρέψουν να εξατομικεύσετε το μήνυμά σας.
