Βοήθεια: Modules

Περιγραφή

Τα modules είναι οι δομές λογισμικού που, εκμεταλλευόμενα την υποδομή που παρέχει το XOOPS, επιτρέπουν στον διαχειριστή του ιστότοπου να προσφέρει στους χρήστες του ιστότοπου δυναμικό ή στατικό περιεχόμενο που δημιουργείται μέσω μιας διεπαφής Ιστοσελίδας, η οποία τις περισσότερες φορές είναι πολύ εύχρηστη.

Μέρος της δύναμης του XOOPS έγκειται στον τρόπο διαχείρισης των modules. Ο διαχειριστής μπορεί εύκολα να εγκαταστήσει, να απεγκαταστήσει, να ενεργοποιήσει ή να απενεργοποιήσει οποιοδήποτε συγκεκριμένο module (εκτός από το module Συστήματος, για προφανείς λόγους). Είναι επίσης δυνατό να μετονομάσετε modules και να καθορίσετε τη σειρά με την οποία θα εμφανίζονται στον τελικό χρήστη στο μενού του ιστότοπου (εάν θέλετε να εμφανίζονται καθόλου), επειδή μπορείτε επίσης να παρέχετε κρυφή λειτουργικότητα, ορίζοντας ενεργά modules που δεν έχουν ορατή παρουσία για τον περιστασιακό χρήστη. Για παράδειγμα, θα μπορούσατε να ορίσετε ένα module ως μη ορατό στο μενού απενεργοποιώντας το εικονίδιο ελέγχου στη στήλη Μενού. Έτσι, για το module Ειδήσεων, θα μπορούσατε στη συνέχεια να στείλετε ιδιωτικά μηνύματα σε επιλεγμένη ομάδα χρηστών σας και να τους πείτε να προσπελάσουν αυτόν τον κατάλογο πληκτρολογώντας τη διεύθυνση στο πεδίο URL, κάτι σαν http://www.yoursite.com/modules/news.

Για να αλλάξετε τη σειρά των modules (που θα αντικατοπτρίζεται στο Μενού), απλώς σύρετε και αποθέστε τα modules στην επιθυμητή θέση.

Η εγκατάσταση module απαιτεί κάποιο σχεδιασμό, τόσο για τα modules που θα εμφανίζονται όσο και για τη διάταξη στις σελίδες. Είναι καλή ιδέα να εξετάσετε άλλους ιστότοπους XOOPS και να δείτε ποια modules χρησιμοποιούνται για την εμφάνιση διαφόρων τύπων περιεχομένου. Αυτό γίνεται εύκολα ανοίγοντας οποιονδήποτε ιστότοπο XOOPS και περνώντας το ποντίκι πάνω από τους συνδέσμους του μενού στις αριστερές και δεξιές πλευρικές μπάρες. Ταυτόχρονα, κοιτάξτε τις διευθύνσεις URL που εμφανίζονται στη γραμμή κατάστασης του περιηγητή σας. Θα πρέπει να δείτε διευθύνσεις όπως αυτή: http://the_site.com/modules/news/.... Όλα τα modules αποθηκεύονται στον κατάλογο /modules/, επομένως ο επόμενος τομέας της διεύθυνσης (σε αυτό το παράδειγμα 'news') δείχνει το module που θα εμφανίσει το περιεχόμενο αυτού του συνδέσμου μενού.

Εάν ανοίξετε τη σελίδα για τον σύνδεσμο, θα δείτε ένα παράδειγμα του πώς χρησιμοποιείται το module και τον τύπο περιεχομένου που εμφανίζει. Η εξέταση άλλων ιστότοπων με αυτόν τον τρόπο θα σας δώσει επίσης μια καλή εικόνα για τις δυνατότητες διάταξης και σχεδιασμού.
